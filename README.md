# 环球旅行计划

这是一个用于规划和记录我的环球旅行的 GitHub 项目。

## 项目结构

本仓库的目的是帮助我（以及可能对环球旅行感兴趣的你！）有条不紊地规划每一个细节，包括：

- **`README.md`**: 项目总览和说明。
- **`global_plan/`**: 存放整个旅行的宏观计划和通用准备清单。
  - `Europe_trip_overview.md`: 欧洲篇总览。
  - `North_America_trip_overview.md`: 北美篇总览。
  - `Northeast_Asia_trip_overview.md`: 东北亚篇总览。
  - `Oceania_trip_overview.md`: 大洋洲篇总览。
  - `1_pre_trip_checklist.md`: 行前准备总清单（护照、保险、疫苗等）。
  - `2_global_visas.md`: 全球签证跟踪和管理。
- **`destinations/`**: 存放每个具体目的地的详细旅行计划。
  - 每个目的地（例如：`Japan/`，`Australia/`）都有一个独立的文件夹。
  - 每个目的地文件夹内包含以下详细文件：
    - `README.md`: 该目的地总览。
    - `0_visa_legal.md`: 签证与当地法规。
    - `0_financial_services.md`: 货币、银行与支付。
    - `0_communications.md`: 通讯与 SIM 卡。
    - `1_attractions.md`: 景点与特色旅游项目。
    - `1_lodging_food.md`: 衣食住行（住宿、餐饮）。
    - `1_local_transportation.md`: 当地交通方案（自驾、公共交通）。
    - `2_international_transportation.md`: 往返交通方案。
    - `3_projects_providers.md`: 服务供应商（向导、租赁等）。
    - `4_notes.md`: 旅行注意事项。
    - `5_itinerary.md`: 详细行程安排。
    - `6_budget.md`: 该目的地预算明细。

## AI 智能规划助手

本项目集成了强大的 AI Agent Skills 和自动化工作流，旨在实现高精度的“一键全流程规划”。

### 🛠️ 专项专家团 (Specialized Agent Squad)

针对规划中的不同环节，我们定义了专属的 AI 专家，以确保数据的准确性和专业深度：

- **Travel Orchestrator (中控台)**: 负责全流程调度与最终 PDF 导出。
- **Visa Consultant (签证顾问)**: 专注签证政策、入境法规及生物安全实务。
- **Logistics Expert (物流专家)**: 负责航司对比、租车保险及路线时间优化。
- **Foodie Expert (美食专家)**: 深度挖掘当地地道风味评价及餐厅分级推荐。
- **Attractions Expert (景点专家)**: 专注地标调研与地理空间聚类分析。
- **Experience Expert (特色项目专家)**: 搜寻硬核体验（实弹射击、极致摄影、冲浪等）及其服务商。
- **Business Inspection (商家研判)**: 负责供应商信用校验、官方预订链接提取及酒店/餐厅细节补全。
- **Itinerary Architect (行程建筑师)**: 基于真实地理逻辑与通行时间编排日程。
- **Budget Wizard (预算向导)**: 基于供应商真实报价进行高精度财务核算。

### 🔄 自动化工作流

- **`/init-destination`**: 一键初始化目的地文件夹及 13 个标准规范文件。
- **`/auto-plan [国家] [天数]`**: **【全流程全自动】**。触发所有专家协作，从调研、合成到生成对应 README，并最终导出 PDF 规划方案。
- **`/research-destination`**: 快速执行基础调研（签证、交通、景点、美食）。
- **`/finalize-plan`**: 针对已有调研数据进行行程合成、预算编制与交付。

# 环球旅行计划 - 整体概览

## 一、大洋洲篇

### 旅行总览

- **总时长**: 90天 (约3个月)
- **预计出发时间**: 2026年2月初
- **主要目的地**: 澳大利亚 (70天：悉尼长居 + 荒漠纵穿自驾)、新西兰 (20天：南北岛深度环线)。
- **核心体验**: 在悉尼深度融入本地生活、挑战 Stuart Hwy 纵穿澳洲红土中心、探索新西兰中土世界的自然奇观。

### 整体路线与时间线概览

- **2026年2月 - 3月中旬**: **悉尼深度探索** (45天)。建立基地，技能提升，筹备后续自驾。
- **2026年3月中旬 - 4月中旬**: **澳洲红土纵穿** (25天)。悉尼 ➜ 墨尔本 ➜ 阿德莱德 ➜ 乌鲁鲁 ➜ 爱丽丝泉。
- **2026年4月中旬 - 5月初**: **新西兰山海环线** (20天)。奥克兰进，深度环绕南北岛精华区域。

## 二、北美篇

### 旅行总览

- **总时长**: 约90天 (3个月)
- **预计出发时间**: 2027年10月1日
- **主要目的地**: 美国 (60天：横跨东西海岸)、加拿大 (30天：东部文化与西部自然)。
- **核心体验**: 从加州阳光出发，横穿美南大陆，探访纽约艺术殿堂；随后北上体验蒙特利尔的法裔文化，最终在温哥华的森林与海洋间收官。

### 整体路线与时间线概览

- **2027年10月 - 11月**: **美国之旅** (60天)。旧金山/硅谷 ➜ 洛杉矶 ➜ 休斯顿 ➜ 佛罗里达 ➜ 亚特兰大 ➜ 华盛顿 ➜ 纽约。
- **2027年11月底 - 12月底**: **加拿大之旅** (30天)。蒙特利尔 ➜ 多伦多 ➜ 跨越大洋飞往温哥华。

## 三、东北亚篇

### 旅行总览

- **总时长**: 约60天 (2个月)
- **预计出发时间**: 2028年4月1日
- **主要目的地**: 日本 (30天：关东、关西、北海道)、韩国 (30天：首尔、釜山、济州岛)。
- **核心体验**: 深度体验日本现代都市与古都韵味的交织，随后转战韩国，感受 K-Pop 文化、地道美食与济州岛的海岛风光。

### 整体路线与时间线概览

- **2028年4月**: **日本之旅** (30天)。东京 ➜ 京都 ➜ 大阪 ➜ 北海道。
- **2028年5月**: **韩国之旅** (30天)。首尔 ➜ 釜山 ➜ 济州岛。

## 四、欧洲篇 (分阶段计划)

### 旅行总览

- **总时长**: 分四阶段，共计约12个月 (每年3个月)
- **预计周期**: 2029年 - 2033年，每年10月至次年2月
- **核心体验**: 历史文明探索、欧洲经典艺术、亚得里亚海之梦、北欧极光奇缘。

### 阶段计划概览

1. **第一年：伊比利亚与法兰西** (2029-2030)。西班牙、葡萄牙、法国。
2. **第二年：中欧经典** (2030-2031)。意大利、瑞士、德国、奥地利。
3. **第三年：南欧与亚得里亚海** (2031-2032)。希腊、克罗地亚、黑山。
4. **第四年：北欧风情** (2032-2033)。丹麦、挪威、瑞典。
